The code places the selfie inside a photo area on that template. You can move or resize that area by editing the four numbers in the JavaScript.

What this version does well

  • Works as a standalone test page.
  • Uses front camera by default on phones.
  • Saves a large, high-quality JPEG file.
  • Supports mobile and desktop camera use.
  • Keeps the booth logic out of your main index page.

Important note

This version creates a branded photo-booth layout. It does not remove the real background behind the person. If you later want the associate cut out and placed in front of a digital background, that would be a separate version using person-segmentation.
